# YourArchive
Проект для участия в конкурсе ITMO.STARS

YourAchiv – бот, который создавался с целью мотивации пользователей. Концепция заключается в том, чтобы отслеживать и сохранять свои ежедневные достижения, а далее использовать эти записи для повышения мотивации в периоды выгорания и недовольства собой. Каждый день пользователь взаимодействует с ботом, записывая свои достижения. Они могут быть как крупными, так и мелкими, например, успешное выполнение заданий на работе или учебе, занятие спортом или обучение новому навыку. Вся информация о достижениях пользователя сохраняется в базе данных. Это позволяет накапливать список своих успехов, к которому можно будет обратиться в будущем. Воспользовавшись кнопкой “Одно из достижений”, бот случайным образом выбирает и показывает любое сохраненное достижение. Также есть возможность вывести полый список. Напоминание о прошлых успехах помогает нам вспомнить о своих возможностях. Это способствует повышению самооценки и мотивации. Бот выступает в роли персонального помощника, который помогает справляться с негативными эмоциями и поддерживает положительное отношение к себе. Целевой аудиторией этого бота являются люди, имеющие склонность к самокритике, испытывающие стресс и периодические выгорания. 
Бот написан на языке программирования Python с использованием библиотек Aiogram 3 и Sqlalchemy. В качестве базы данных использовалась SQLiteStudio.
Касательно будущего развития проекта, можно рассмотреть внедрение новых функций:
1.  Возможность устанавливать краткосрочные и долгосрочные цели, бот будет отслеживать прогресс и мотивировать на их достижение. 
2.  Возможность настройки напоминаний для записи достижений, чтобы пользователь вносил новые записи ежедневно.
3.  Возможность в удобном формате создавать челленджи на любой срок. Например, выходить на пробежку каждое утро в течение недели.
Проект направлен на создание инструмента, который будет помогать пользователям справляться с выгоранием и негативными мыслями через напоминание о их прошлых достижениях. Данный бот является уникальным, поскольку изначально создавался для меня. Я искала подобные реализации моей идеи, но ничего не нашла. Поэтому было принято решения самостоятельно сделать нужного мне бота. В данный момент я и мои друзья используют мой продукт.
